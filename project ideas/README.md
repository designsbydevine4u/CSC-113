# Project Ideation - Build my Business

## Idea 1: Calendar/Reminders

**Problem/Goal:**
Being a Graphic Designer will require some research and to be a business owner will require maintenance on media, i.e. social media pages etc. It would be nice to have assistance with keeping up with these.

**AI Tool(s):**
Not sure but maybe a Bot that can run a Calendar would be nice

**Why This Interests Me:**
I have a horrible memory and having something that can "think" for me and add things to my schedule without me having to remember would solve some big issues.

**Rough Scope:**
Maybe a couple of months or so to get it started and figure out exactly what I need it to track for me.
